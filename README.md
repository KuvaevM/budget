# Бот для контроля личных финансов на языке Python


## Приступая к работе
* Установите все библиотеки, перечисленные в requirements.txt
* Заполните файл конфигурации settings.ini. Он должен содержать в себе 2 токена, один для работы с телеграм ботом, второй для работы с vk-api.
 Подойдет как личный вк токен, так и токен для работы с вк приложением (для безопасности рекомендуется использовать второй).
* Запустить файл allocator.py. После запуска создастся база данных по адресу data_base/budget.db, работа программы будет логироваться. Лог можно найти в файле inform.log.
